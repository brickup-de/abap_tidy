The suggestions on this site are [optimized for reading, not for writing](/clean-code/formatting/optimize-for-reading-not-for-writing/).
As the ABAP Formatter doesn't cover them, some of them produce additional manual work to reformat statements
when name lengths etc. change; if you want to avoid this, consider dropping rules like
[Align assignments to the same object, but not to different ones](/clean-code/formatting/align-assignments-to-the-same-object-but-not-to-different-ones/).
